# Website

This website is built using [Blume](https://useblume.dev/), a markdown-first documentation framework.

Requires Node.js 22.12 or newer.

### Installation

```
$ npm install
```

### Local Development

```
$ npm run dev
```

This command starts a local development server at http://localhost:4321. Most changes are reflected live without having to restart the server.

### Build

```
$ npm run build
```

This command generates static content into the `dist` directory, which can be served by any static content hosting service.
